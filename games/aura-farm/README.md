Incremental Aura Farming

Open `index.html` in a browser.

This build includes:
- mobile-safe portrait layout with in-stage UI placement
- blurred stretched desktop background behind the centered stage
- embedded `Cormorant-Regular.ttf`
- fixed right-side boost column under the hamburger menu
- seven boost types with always-visible icon and stat text
- disabled boosts shown by faded icons only, without blocker overlay icons
- clearer boost labels such as `-1s chill timer`, `+20% chill reward`, and `+2% chance`
- random boost rolls that usually land near baseline but sometimes spike higher
- chill reward is steady: base payout is max(10 aura, 1% current aura), multiplied by chill reward upgrades
